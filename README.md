📖 Sobre o Projeto

O Sorteador de Números é uma aplicação web interativa em que o usuário tenta descobrir um número secreto gerado aleatoriamente pelo algoritmo.

O projeto foi desenvolvido com o objetivo de fortalecer habilidades fundamentais do desenvolvimento front-end, trabalhando conceitos como:

Estruturas condicionais
Funções
Manipulação do DOM
Eventos
Geração de números aleatórios
Atualização dinâmica da interface
Responsividade

Além da lógica da aplicação, também houve uma preocupação com a construção de uma interface simples, intuitiva e agradável para o usuário.

✨ Funcionalidades

✅ Geração aleatória de números
✅ Validação de tentativas do usuário
✅ Feedback visual durante o jogo
✅ Interface intuitiva e responsiva
✅ Atualização dinâmica de elementos na tela
✅ Experiência interativa e fluida
